Principais Projetos

Impressão 3D de Estruturas Anatômicas

Os laboratorios IFMaker são equipados com impressoras 3D capazes de reproduzir estruturas anatômicas com alta fidelidade, para fins de estudo e outros.

Microestações meteorológicas

O Projeto IFMaker contempla a instalação e operação de Microestações meteorológicas para monitoramento de microclimas urbanos

Projetos de Automação

Uma das principais atividades do laboratorio IFMaker é o desenvolvimento e automação de robos para diferentes funções.
